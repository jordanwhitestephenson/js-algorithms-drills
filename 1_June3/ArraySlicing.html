Array Slicing

Array slicing involves taking a subset from an array and allocating a new array with those elements

var myArray = ['Jordan', 'Mike', 'Bob', 'Kitty']

How would we get just 'Mike'?

Well, our starting index would be 1 ("Mike"), and then our ending index would be Mike with an index of 2

var myArray = ['Jordan', 'Mike', 'Bob', 'Kitty'] var slicedArray = myArray.slice(1, 2) console.log(slicedArray) ===> output would be ["Mike"]
